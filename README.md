# 国家·辽宁·大连公开指标查询分析系统

公开统计数据与经确认可公开的副省级城市对标数据查询分析系统，适合快速完成国家、辽宁省、大连市及对标城市之间的指标检索、横向对比、历年趋势查看和材料文字摘录。

## 在线访问

GitHub Pages 发布后访问：

`https://xujiann.github.io/health-indicators/`

## 功能

- 关键词检索：支持按指标名、对标口径、地区、备注搜索。
- 专题入口：支持一键切换全国卫健公报、卫生资源对标、大连趋势、统计公报补录等常用视角。
- 多级筛选：支持全国、省区市、副省级城市层级，以及类别、子类、年份筛选。
- 对比分析：自动识别横向对比、历年趋势、目标约束类指标，并生成图表卡片。
- 明细列表：保留逐条数据表，便于核验来源、年度、单位和备注。
- 材料复用：卡片支持一键复制为材料文字。

## 数据内容

- `index.html`：可直接托管到 GitHub Pages 的静态查询分析系统，数据已内嵌，离线也可打开。
- `公开指标数据库.xlsx`：结构化公开数据，共 2350 条、19 个字段，包含全国近十年卫生健康统计公报核心序列、2022-2024 年公报扩展分类指标、1998-2025 年全国经济人口卫生医保图片长序列、2025 年国家统计局卫生资源年度公报快报、国家医保局统计快报及 15 个副省级城市对标数据。
- `data/national-economy-pop-health-insurance-additions.json`：根据用户提供图片表结构化录入的 1998-2025 年全国经济、人口、卫生、医保相关长序列数据；空白单元未录入，正式引用需复核原始来源。
- `data/national-health-bulletin-additions.json`：国家卫健委 2015-2024 年卫生健康统计公报核心指标补录数据。
- `data/national-health-bulletin-expanded-additions.json`：国家卫健委 2022-2024 年卫生健康统计公报 PDF 表格扩展指标，按资源、人员、费用、服务、运行、疾控、妇幼、中医药等分类。
- `data/national-official-2025-health-insurance-additions.json`：国家统计局《2025 年国民经济和社会发展统计公报》卫生资源、服务量指标，以及国家医保局《2025 年医疗保障事业发展统计快报》参保、基金、待遇、生育、长护和异地结算指标；保留独立对比口径，便于与后续国家卫健委 2025 年公报核验。
- `data/stat-bulletin-additions.json`：相关城市 2024 年官方统计公报补录数据。
- `data/subprov-stat-bulletin-additions.json`：副省级城市 2023 年官方统计公报核心指标补录数据。
- `docs/数据字段说明.md`：字段口径和维护说明。
- `docs/统计公报补录说明.md`：统计公报补录范围、来源索引与维护原则。
- `docs/官方数据源巡检.md`：官方数据源巡检范围、自动检测边界和人工复核入库流程。
- `docs/发布与隐私边界.md`：公开发布边界和安全检查项。

## 数据范围与来源

本仓库含各级政府官网正式发布的统计公报、财政公开数据，以及经确认可公开的副省级城市对标数据，覆盖经济、人口、财政、卫生健康等指标。

本版已补入国家卫生健康委 2015-2024 年《我国卫生健康事业发展统计公报》核心序列，并对 2022-2024 年官方 PDF 中可稳定结构化的表格指标做扩展入库，覆盖卫生资源、床位资源、卫生人员、卫生费用、医疗服务、医院运行、基层卫生服务、中医药服务、疾病控制与公共卫生、妇幼与公共卫生、医疗质量与服务、健康结果等分类。由于截至本次核验未发现国家卫健委发布 2025 年同名统计公报，近十年序列按 2015-2024 年处理。

另补入国家统计局 2025 年年度公报中的卫生资源与服务量，以及国家医保局 2025 年统计快报中的参保、基金和待遇指标。两组记录均保持来源专属的对比键，不把年度快报与国家卫健委同名统计公报视为同一口径。每周自动巡检官方栏目是否更新；发现新材料后仍需完成字段、单位和可比性复核，才会进入公开数据库。

城市侧已补入南京、广州、深圳、武汉、济南、沈阳、大连等城市 2024 年官方统计公报核心指标，并补入南京、广州、深圳、武汉、济南、沈阳 2023 年官方统计公报核心经济、人口、财政指标；副省级城市横向对标数据已按可公开口径处理。引用和正式使用时，请以原始公报或正式来源为准。
