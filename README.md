# 国家·辽宁·大连公开指标查询分析系统

各级政府公开发布统计与财政数据的交互式查询分析系统，适合快速完成国家、辽宁省、大连市及对标城市之间的指标检索、横向对比、历年趋势查看和材料文字摘录。

## 在线访问

GitHub Pages 发布后访问：

`https://xujiann.github.io/health-indicators/`

## 功能

- 关键词检索：支持按指标名、对标口径、地区、备注搜索。
- 多级筛选：支持全国、省区市、副省级城市层级，以及类别、子类、年份筛选。
- 对比分析：自动识别横向对比、历年趋势、目标/约束类指标，并生成图表卡片。
- 明细列表：保留逐条数据表，便于核验来源、年度、单位和备注。
- 材料复用：卡片支持一键复制为材料文字。

## 数据内容

- `index.html`：可直接托管到 GitHub Pages 的静态查询分析系统，数据已内嵌，离线也可打开。
- `公开指标数据库.xlsx`：结构化公开数据，共 86 条、19 个字段。
- `docs/数据字段说明.md`：字段口径和维护说明。
- `docs/发布与隐私边界.md`：公开发布边界和安全检查项。

## 数据范围与来源

本仓库仅含各级政府官网正式发布的统计公报与财政公开数据，涵盖经济、人口、财政、卫生健康等公开指标。

来源包括国家统计局、财政部，辽宁省、大连市、上海市、沈阳市统计局及政府公开统计公报。引用和正式使用时，请以原始公报为准。

## 重要边界

本公开仓库不包含任何未公开文件、征求意见稿、内部统计提要或内部收集表数据。全量内部数据请只放在私有仓库中维护，且不要为私有仓库启用 GitHub Pages。
