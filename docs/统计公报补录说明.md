# 统计公报补录说明

本文件记录公开库中通过官方统计公报补录的数据范围，便于后续按城市和年度继续扩展。

## 当前补录范围

- 时间范围：本轮先补入 2024 年官方统计公报中可稳定核验的核心指标。
- 城市范围：南京、广州、深圳、武汉、济南、沈阳、大连。
- 指标范围：GDP、人均 GDP、常住人口、户籍人口、城镇化率、地方一般公共预算收入、一般公共预算支出，以及部分公报明确披露的卫生健康指标。
- 数据文件：`data/stat-bulletin-additions.json`。

## 维护原则

- 只补录官方统计局、政府门户或其可追溯公报索引中的数据。
- 搜索结果、百科、转载站点仅用于发现线索，不作为公开库取数来源。
- 同一城市同一年度同一指标如已有经确认可公开数据，生成脚本按地区、年份、指标、性质、层级去重，公报补录可覆盖同口径空源或旧源。
- 后续扩展 2020 年以来历史序列时，优先补齐 15 个副省级城市的 GDP、人口、财政三类核心指标，再扩展卫生资源与服务指标。
