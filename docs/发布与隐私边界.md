# 发布与隐私边界

本仓库定位为公开在线系统，可开启 GitHub Pages。发布前应确认：

- 只包含公开统计公报、财政公开数据和政府官网已发布内容。
- 不包含内部资料、不予公开文件、征求意见稿、内部统计提要、内部收集表。
- `index.html` 内嵌数据与 `公开指标数据库.xlsx` 保持同一公开口径。
- 私有全量数据只进入 `health-indicators-private` 等私有仓库，不开启 Pages。

建议发布前执行敏感词扫描：

```powershell
rg -n "内部资料|不予公开|征求意见稿|统计提要|数据收集表" .
```

如果扫描命中公开页面或公开 Excel 中的真实数据内容，应先移除或改投私有仓库后再发布。
