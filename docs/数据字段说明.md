# 数据字段说明

公开指标数据库共 432 条记录、19 个字段，含经确认可公开的 15 个副省级城市对标数据。字段设计与全量库保持一致，便于后续从公开来源继续补录。

| 字段 | 含义 |
| --- | --- |
| `region_code` | 行政区划或地区编码 |
| `region` | 地区名称 |
| `level` | 地区层级，如国家、省、市 |
| `year` | 指标年度 |
| `category` | 一级类别，如经济、人口、财政、卫生健康 |
| `subcategory` | 二级类别 |
| `indicator` | 原始指标名称 |
| `nature` | 指标性质，如实际值、目标值、约束值 |
| `value` | 数值 |
| `unit` | 单位 |
| `yoy` | 同比或增减说明 |
| `deadline` | 完成期限，公开统计数据通常为空 |
| `responsible` | 责任主体，公开统计数据通常为空 |
| `source` | 来源名称 |
| `doc_no` | 文号或公开公报说明 |
| `source_url` | 来源链接 |
| `note` | 备注 |
| `compare_key` | 对比归并口径 |
| `region_tier` | 查询系统使用的地区层级 |

维护时优先补充政府官网、统计局、财政部门等公开来源。新增记录后，需要同步更新 `index.html` 内嵌的 `DATA` 数据。
