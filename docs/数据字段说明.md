# 数据字段说明

公开指标数据库共 2584 条记录、19 个字段，包含全国近十年卫生健康统计公报核心序列、2022-2024 年国家卫健委统计公报扩展分类指标、1998-2025 年全国经济人口卫生医保图片长序列、2023-2025 年国家医保局年度统计公报与统计快报专题、2025 年国家统计局卫生资源年度公报快报，以及经确认可公开的 15 个副省级城市对标数据。字段设计与全量库保持一致，便于后续从公开来源继续补录。

| 字段 | 含义 |
| --- | --- |
| `region_code` | 行政区划或地区编码 |
| `region` | 地区名称 |
| `level` | 地区层级，如国家、省、市 |
| `year` | 指标年度 |
| `category` | 一级类别，如经济、人口、财政、卫生健康 |
| `subcategory` | 二级类别 |
| `indicator` | 原始指标名称 |
| `nature` | 指标性质，如实际值、目标值、约束值 |
| `value` | 数值 |
| `unit` | 单位 |
| `yoy` | 同比或增减说明 |
| `deadline` | 完成期限，公开统计数据通常为空 |
| `responsible` | 责任主体，公开统计数据通常为空；全国卫生健康公报补录为国家卫生健康委 |
| `source` | 来源名称 |
| `doc_no` | 文号或公开公报说明 |
| `source_url` | 来源链接 |
| `note` | 备注 |
| `compare_key` | 对比归并口径 |
| `region_tier` | 查询系统使用的地区层级 |

## 维护方式

- 新增公开补录数据优先写入 `data/*-additions.json`，再运行 `scripts/update-public-data.mjs` 统一生成 Excel 和页面内嵌数据。
- 页面将 `responsible`、`source`、`doc_no` 和 `source_url` 归并为发布部门筛选标签：国家医保局、国家卫健委、国家统计局、财政部、地方部门、用户提供材料和其他公开来源。该标签仅用于检索聚合，不替代原始责任主体字段。
- 全国卫生健康序列采用紧凑 JSON 结构维护，脚本会自动展开为标准 19 字段记录。
- 城市和其他地区数据仍可直接使用标准 19 字段数组维护。
- 维护时只使用政府官网、统计局、财政部门、国家卫生健康委等公开来源；搜索结果或转载站点只作为发现线索。
