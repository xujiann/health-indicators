# 国家医保数据专题

## 范围

专题收录全国层面的公开医疗保障数据，当前包括三类序列：

| 序列 | 年份 | 用途 |
| --- | --- | --- |
| 国家医保局医保数智库待遇长序列 | 2012-2025 | 观察待遇享受、医疗费用、异地就医等长期变化 |
| 全国医疗保障事业发展统计公报 | 2023-2025 | 最终统计口径，用于年度横向和纵向比较 |
| 医疗保障事业发展统计快报 | 2023-2025 | 初步统计口径，保留发布时点和早期观察价值 |

## 指标分类

- 参保规模：基本医保、职工医保、居民医保及生育保险参保人数、参保率。
- 基金收支：基本医保、职工医保、统筹基金、个人账户、居民医保的收入、支出和结存。
- 待遇人次与费用：职工和居民的门诊、慢特病、住院、药店购药、次均住院费用、目录内支付比例。
- 生育保险与医疗救助：待遇享受、基金支出、资助参保、门诊及住院救助。
- 药品目录与采购：目录药品数量、新纳入品种、谈判药报销人次。
- 异地就医与经办服务：异地就医人次和费用、跨省直接结算、个人账户共济。
- 基金监管与长期护理保险：追回基金、长期护理保险参保、待遇、基金、服务机构和人员。

## 口径规则

同一统计年度的快报与年度统计公报可能因统计时点、四舍五入或最终核定而不同。系统使用 `compare_key` 的来源后缀区分二者：

- `（年度统计公报）`：用于 2023-2025 年最终口径趋势卡。
- `（统计快报）`：保留初步统计数，不覆盖年度公报。
- `（医保待遇专题）`：医保数智库长序列，按其附件口径独立比较。
